# Datree Helm Plugin

A Helm plugin to validate charts against the Datree policy

## Installation
```
helm plugin install https://github.com/datreeio/helm-datree
```

## Usage

### Test against the default policy
```
helm datree test CHART_DIRECTORY
```

### Check plugin version
```
helm datree version
```

### See help text
```
helm datree help
```
